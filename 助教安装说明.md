# 龙门计划 - 助教安装说明

## 第一步：填写你的名字

打开 `config.json`，把 `请填写助教名称` 改成你自己的名字，例如：

```json
{
  "ta_name": "王晓明"
}
```

---

## 第二步：安装自动上传

### Mac 用户

打开终端，进入文件夹，运行：

```bash
bash install_mac.sh
```

安装完成后，每天 23:00 会自动把当天内容上传。

### Windows 用户

右键 `install_windows.bat`，选择 **以管理员身份运行**。

安装完成后，每天 23:00 会自动把当天内容上传。

---

## 第三步：确认 Git 已配置

自动上传需要 Git 提前登录你的 GitHub 账号。

**检查方法**：打开终端/命令提示符，运行：

```bash
git config user.name
git config user.email
```

如果没有输出，运行以下命令（替换成你的信息）：

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

---

## 日常使用

**不需要做任何操作。** 每天 23:00 脚本自动运行。

如需查看上传日志：

- Mac：查看 `.longmen/auto_push.log`
- Windows：查看 `.longmen\auto_push.log`

---

## 常见问题

**Q：上传失败怎么办？**
查看日志文件，截图发给管理员。

**Q：想手动上传怎么办？**
- Mac：`bash auto_push_mac.sh`
- Windows：双击 `auto_push_windows.bat`
